>>>>>>>>>>>>> Data Types
Name        | Description
VarInt      | Variable length interger, stops reading on MSB
String      | UTF-8 encoded string of VarInt length
Position    | 4 bytes, 0bXXXXXXXXXXXXXXXXYYYYYYYYYYYYYYYY, 2 bytes for x, 2 bytes for y
Boolean     | 1 byte, 0x00 or 0x01
Bit Flags   | 1 byte, 0b00000000

>>>>>>>>>>>>> Serverbound packets
Name        | IDs  | Fields
Join        | 0x00 | Lobby (VarInt), Name (string)
Move        | 0x01 | Position

>>>>>>>>>>>>> Clientbound packets
Name        | IDs  | Fields
Join accept | 0x00 | None
Join deny   | 0x01 | Reason (string)
Upd Players | 0x02 | Length (VarInt), [Name, Position]

>>>>>>>>>>>>> Notes
Clientbound 0x02 packet repeats [Name, Position] the VarInt length
